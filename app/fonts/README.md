# Paperlogy

Paperlogy web font files are bundled locally so the app does not depend on a runtime font CDN request.

- Source: `https://github.com/fonts-archive/Paperlogy`
- Family: `Paperlogy`
- Format: `woff2`
- Weights: 100, 200, 300, 400, 500, 600, 700, 800, 900
- License: SIL Open Font License according to the upstream README.
